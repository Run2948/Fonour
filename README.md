# Asp.Net Core 权限管理系统

## Code First 操作步骤
* 使用程序包管理器控制台(前提安装 ...Sqlsever、...SqlSever.Design )
  * `Add-Migration Init`
  * `Update-DataBase`
  * `Remove-Migration `
* 使用 .NET Core CLI 命令
  * `dotnet ef migrations add init`
  * `dotnet ef database update`*
